### 常用方法

| 方法名             | 说明                                               |
| ------------------ | -------------------------------------------------- |
| length()           | 获取字符串长度                                     |
| charAt(i)          | 获取第几个位置上的字符                             |
| equal()            | 判断两个字符串是否相等；==判断两个基本数据类型相等 |
| equalsIgnoreCase() | 忽略大小写比较相等，常用于判断验证码               |
| toLowerCase()      | 转换成小写字母                                     |
| toUpperCase()      | 转换成大写字母                                     |
| startsWith()       | 判断字符串是不是以什么开头                         |
| endsWith()         | 判断字符串是不是以什么结尾                         |
| concat()           | 字符串拼接                                         |
| indexOf()          | 字符串查找                                         |
| lastIndexOf()      | 字符串查找，从后面开始找                           |
| substring()        | 字符串截取                                         |
| trim()             | 去掉字符串两边的空格                               |
| split()            | 字符串拆分                                         |
| replace(old,new)   | 字符串替换                                         |
## 包装类
- 基本数据类型所对应的类，具有属性和方法

| 基本数据类型 |  包装类   |
| :----------: | :-------: |
|     byte     |   Byte    |
|    short     |   Short   |
|     int      |  Integer  |
|     long     |   Long    |
|    float     |   Float   |
|    double    |  Double   |
|     char     | Character |
|   boolean    |  Boolean  |
- 常用方法，各种数据类型之间的转换
## Character的方法：
* toCharArray(): 字符串转换为字符数组
* isLetter()/isDigit()/isSpaceChar(), 判断是否为字母、数字或者空格
