    List:
         特性,ArrayList (数组列表)        ,LinkedList (链表)
      底层实现,动态数组 (连续内存)          ,双向链表 (节点指针连接)
      访问速度,快 O(1) (支持按索引直达)     ,慢 O(n) (需从头遍历)
      增删速度,慢 (中间增删需移动后面所有元素),快 (只需改变指针指向)
      适用场景,读多写少，通过下标访问,频繁在头部或中间插入/删除

      list.add(E e);          // 添加元素到末尾
      list.get(int index);    // 获取指定位置元素 (ArrayList强项)
      list.size();            // 获取长度
      list.remove(int index); // 删除指定位置元素
      list.contains(Object o);// 判断是否包含某个元素
      addAll(Collection c);   //并集: 把另一个集合的所有元素加进来。
      removeAll(Collection c);//差集: 删除那些也存在于另一个集合中的元素（即：从 A 中去掉所有 B 有的）。
      retainAll(Collection c);//交集: 只保留那些同时也存在于另一个集合中的元素（即：留下 A 和 B 都有的）。
      indexOf(Object o) ;     // lastIndexOf(Object o): 查找某个元素第一次或最后一次出现的索引位置。如果找不到返回 -1。
      subList(int fromIndex, int toIndex)//截取列表的一部分（视图）。(注意： 对 subList 的修改会直接影响原列表！)
      set(int index, E element): 替换指定位置的元素。
      sort(Comparator c): 直接对列表进行排序（Java 8+）。
      toArray(): 把集合转换成普通数组


    Set 特点是不允许元素重复。
    HashSet: 最常用。无序，速度最快（基于哈希表）。
    TreeSet: 有序（默认升序）。速度稍慢，但能自动排序。
    set.add(E e);           // 添加元素 (如果已存在，返回false)
    set.remove(Object o);   // 删除元素
    set.contains(Object o); // 判断是否存在 (速度极快)
    set.size();             // 获取大小


    Map:
    Key (键): 不可重复
    Value (值): 可以重复
    map.put(K key, V value);  // 添加或修改键值对
    map.get(Object key);      // 根据键获取值
    map.containsKey(K key);   // 判断键是否存在
    map.remove(K key);        // 根据键删除映射
    map.keySet();             // 获取所有键的集合 (返回一个Set)
